What is Omnibase?

Omnibase is an opensource, 4 wheel drive robot. The robot behaves as an omniwheeled robot and can move in any direction in X-Y plane. For simplicity the wheels have been modelled as cylinders, which are giving us same results as an omniwheel.

* Link to out github repo: https://github.com/ERC-BPGC/omnibase
